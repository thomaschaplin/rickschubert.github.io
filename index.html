I created a mobile app which periodically crawls ebay for auctions that match products the user is looking for. Via push notifications, the user is being notified of auctions that will end soon and which are below the maximum amount they are willing to pay.

  • Backend written in Go
  • Mobile application written in React Native, works for Android and iOS devices
  • Serverless AWS stack: Lambda, DynamoDB, API Gateway, CloudWatch

Acquisition Workflow

My client needed to automate and structure a complex workflow involving several thousand interconnected data sets which had to be consistently updated by colleagues and partners. I created:

  • Node.JS REST API with several routes to query and update entries
  • MongoDB database
  • Homepage showing up-to-date workflow statistics as diagrams
  • Table-style dashboard to browse through the dataset
  • Modal overlay to update existing data
  • Heroku deployment
